SweepLine.py

Synopsis

Voronoi tessellation using the sweep-line algorithm.

Description

SweepLine.py is a stand-alone Voronoi diagram construction program, which also contribute as an essential part of EXSdetect. In this program, the sweep-line algorithm developed by Steven Fortune in 1986 is implemented with pure Python code. A few additional functions are also provided.

Syntax

  SweepLine.py inputfile [options]

Dependence

You need to have Python in your system, together with a few commonly used Python packages: 

  numpy    numerical python package
  pyfits   interface to FITS formatted files
It has only been tested with Python 2.7.
